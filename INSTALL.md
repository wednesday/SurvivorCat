# 用户安装指南

## macOS 用户

### 如果提示"应用已损坏，无法打开"

**不要担心！** 这不是应用真的损坏了，这是macOS的安全保护机制。

#### 解决方法（选择一种）：

**方法1：使用终端命令（推荐）**

1. 打开"终端"应用（在"应用程序 > 实用工具"中）
2. 复制并运行以下命令：

```bash
# 如果应用在"下载"文件夹
cd ~/Downloads
xattr -cr "Survivor Cat.app"

# 如果应用在"应用程序"文件夹
cd /Applications
sudo xattr -cr "Survivor Cat.app"
```

3. 现在可以正常打开应用了

**方法2：右键打开**

1. 在Finder中找到应用
2. 按住 **Control** 键，点击应用图标
3. 选择"打开"
4. 在弹出的对话框中再次点击"打开"按钮

**方法3：系统设置**

1. 尝试打开应用（会被阻止）
2. 打开"系统设置" > "隐私与安全性"
3. 在底部找到被阻止的应用提示
4. 点击"仍要打开"

### 为什么会这样？

macOS 有一个叫"Gatekeeper"的安全功能，它会检查应用是否由已知开发者签名。由于这是一个独立开发的免费游戏，没有付费购买Apple开发者证书（$99/年），所以会被拦截。

这是完全正常的，很多免费开源软件都会遇到这个问题。

## Windows 用户

### 如果提示"Windows保护了你的电脑"

1. 点击"更多信息"
2. 点击"仍要运行"

这是Windows SmartScreen的保护机制，因为应用没有购买代码签名证书。

## Linux 用户

Linux通常不会有这些限制，直接运行即可：

```bash
# 给予执行权限（如果需要）
chmod +x survivor-cat

# 运行
./survivor-cat
```
